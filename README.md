Требования к рабу
1 Сделать систему инвентаря
То есть: При взятие обьекта в руку он отправится в инвентарь(в скрипте Throvable так можно) инвентарь будет в виде GUI или как нибудь еще(например при нажатие кнопки открывался рюкзак)
При выборе слота предмет появлялся в правой руке
2 Сделать скрипт взаимодействия:
То есть: когда у тебя выбран предмет и ты касаешся(или наводишь и нажимаешь курок) правой рукой обьекта с этим скриптом происходит какое-то действие
3 Проверка целостности
То есть: При открытии ящика, двери, включение света и тд. действия куда-то записываются
И при проверке(маньяком) будет маньяк ругатся(случайно выбирает неправильный предмет)
